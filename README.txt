Bienvenidos a este proyecto. Estaré haciendo un simulador o historial de tweets con ayuda del Local Storage.

Donde el usuario podrá escribir en un texarea algún comentario y cuando el usuario le de click al botón submit,
este comentario se almacenara en una sección de nuestrá página. Tambieén el usuario tendrá la opción de eliminar 
la n cantidad de tweets que tenga almacenados en su local storage.

Para esto etaré utilizando local storage y darle la lógica de programación en JavaScript
